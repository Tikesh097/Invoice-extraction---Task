# Test Invoices

Place sample invoice files here for testing the application.

## Recommended Test Files
- Standard US invoice (PDF)
- Indian GST invoice (JPG/PNG)
- EU invoice with multiple line items (PDF)
- Handwritten or scanned invoice (low quality test)

## Where to Get Sample Invoices
- Use any real invoice PDF/image you have
- Generate sample invoices at: https://invoice-generator.com
- Use publicly available invoice samples from Kaggle datasets

## Testing Notes
- Upload via the Upload page
- Check confidence score after extraction
- Try uploading the same invoice twice to test duplicate detection
- Upload two invoices from the same vendor to test format learning
